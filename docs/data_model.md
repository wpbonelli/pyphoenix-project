# Data Model



## Overview

MODFLOW 6 input files are structured into one or more blocks.

### Blocks

A block is a collection of related settings in a MODFLOW 6 input file. A block is delimited by lines containing `BEGIN <block name>` and `END <block name>`.

A block contains zero or more options identified by keywords. Each setting contains one of three forms of input data: scalar, array, or table.
